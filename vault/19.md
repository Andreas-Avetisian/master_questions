# Describe the difference between supervised and unsupervised learning. What algorithms are typical examples for each method? Name some use cases.

**Supervised learning**

- Uses **labeled data**: for each training example, the correct output / class is already known.
- Goal: learn a relationship between input features and a target value so the model can make **predictions** for new data.
- Typical tasks: **classification** and **regression**.
- Typical examples of algorithms: **linear regression**, **logistic regression**, **decision trees**, **k-nearest neighbor**, **neural networks**.
- Typical use cases: **credit scoring**, **churn prediction**, **campaign management**, **customer lifetime value prediction**, **price or load forecasting**, **spam detection**.

**Unsupervised learning**

- Uses **unlabeled data**: class memberships / target values are **not available**.
- Goal: discover structure in the data, for example by grouping similar objects.
- Typical tasks: **clustering**, **association analysis**, **dimensionality reduction**, **anomaly detection**.
- Typical examples of algorithms: **k-means**, **hierarchical clustering**, **expectation maximization**, **principal component analysis (PCA)**, **self-organizing maps**.
- Typical use cases: **customer segmentation**, **market basket analysis**, **customer journey analysis**, **grouping similar documents or products**, **anomaly detection**.

**Main difference**

- **Supervised learning** learns from examples with the correct answer already given.
- **Unsupervised learning** works without predefined labels and tries to find hidden patterns or clusters in the data.

**Short exam answer**

Supervised learning uses labeled training data and is mainly used for prediction tasks such as classification and regression. Unsupervised learning uses unlabeled data and is mainly used to discover structure in data, especially clusters or other patterns. Typical supervised methods are linear/logistic regression, decision trees, k-NN, and neural networks; typical unsupervised methods are k-means, hierarchical clustering, EM, and PCA. Common use cases are credit scoring or churn prediction for supervised learning, and customer segmentation or market basket analysis for unsupervised learning.
