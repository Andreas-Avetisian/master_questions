# Name the 5 cost types in a typical research project. Give an example on how to calculate personal costs in a typical research project.

Typical research projects distinguish **5 cost types**:

- **Personal**
- **Travel costs / open access fees**
- **Third party costs**
- **Acquisition**
- **Overhead** (`AFA`)

## Example: Personal Cost Calculation

The cited slide gives the usual formula:

`(Annual wage) * Overhead / Divider`

Typical values from the lecture are:

- **Divider:** `1720 h/year`
- **Overhead:** usually `20-25%`

Example with an annual wage of `EUR 60,000` and an overhead factor of `1.25`:

`(60,000 * 1.25) / 1720 = 43.60 EUR/h`

If that person works `100` project hours, the personal cost is:

`43.60 * 100 = 4,360 EUR`

The slide also notes that in practice the **real hourly rates** must finally be calculated.
