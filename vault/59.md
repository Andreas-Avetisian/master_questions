# Organisational Security: Identifying a user: Name and explain three ways of identifying a user. Give examples for each.

There are three classic ways of identifying or authenticating a user.

- **Something the user knows** (`what you know`): the user proves identity by knowing secret information. Examples: password, PIN, graphical password, secret question.
- **Something the user is** (`what you are`): the user is identified by biometric characteristics. Examples: fingerprint, face recognition, iris scan, voice.
- **Something the user has** (`what you have`): the user proves identity by possessing a specific object or token. Examples: chip card, RFID transponder, passport, USB token, OTP device or authenticator app.

These factors are often combined in **multi-factor authentication**, for example a bank card plus PIN, or a password plus fingerprint.
