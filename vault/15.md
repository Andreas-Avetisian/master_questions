# Describe the relationship between AI and Machine Learning. What are the different fields of Machine Learning and how do they differ?

Artificial Intelligence (AI) is the broader field of building systems that perform tasks that normally require human intelligence. Machine Learning (ML) is a subset of AI: instead of a human defining all rules explicitly, an ML model learns patterns and rules from sample data. Deep Learning is a further subset of ML.

In short:
- AI is the umbrella term.
- ML is one important approach within AI.
- Deep Learning is a special area within ML.

## Relationship Between AI and ML

The basic difference from classic programming is:
- Classic programming: a human defines the rules.
- Machine learning: the model learns the rules from data.

That is why ML is used for problems where the rules are hard to write manually but enough good data is available.

## Main Fields of Machine Learning

| Field | Data / feedback | Main goal | Typical result |
| --- | --- | --- | --- |
| Supervised learning | Labeled data | Predict a known target | Classification or regression |
| Unsupervised learning | Unlabeled data | Find hidden structure in data | Clusters, associations, patterns |
| Reinforcement learning | Reward / penalty from interaction | Learn the best strategy over time | A policy that maximizes reward |

### 1. Supervised Learning

Supervised learning uses labeled examples, meaning the correct output is known during training. The model learns the relationship between input features and the target value.

Typical tasks:
- Classification: predict a category.
- Regression: predict a numeric value.

Important idea:
- The model is trained on training data and checked on test data.

### 2. Unsupervised Learning

Unsupervised learning uses data without predefined labels. The model is not told what the correct answer is, so it tries to detect structure on its own.

Typical tasks:
- Clustering similar data points.
- Finding associations or patterns in the data.

Main difference from supervised learning:
- There is no known target output.

### 3. Reinforcement Learning

Reinforcement learning is based on an agent interacting with an environment. The agent takes actions and receives rewards or penalties. Over time, it learns a strategy that maximizes long-term reward.

Main idea:
- Learning happens through trial and error, not through fixed labels.

## Key Differences

- Supervised learning learns from correct answers.
- Unsupervised learning learns from structure in unlabeled data.
- Reinforcement learning learns from rewards while acting in an environment.

## Short Exam Answer

AI is the broad field of intelligent systems, while machine learning is a subset of AI in which models learn rules from data instead of being fully programmed by humans. The main fields of ML are supervised learning, unsupervised learning, and reinforcement learning. They differ mainly in the type of feedback: supervised learning uses labeled data, unsupervised learning uses unlabeled data, and reinforcement learning uses rewards and penalties from interaction with an environment.
